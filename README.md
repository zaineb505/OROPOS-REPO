# OROPOS - Site Web de Point de Vente (POS)

**Projet réalisé dans le cadre de ISS4U | Juil 2024 - Août 2024**

OROPOS est un site web de point de vente (POS) inspiré des solutions professionnelles comme OROPOS. L’objectif était de fournir une **interface utilisateur intuitive, réactive et performante**, adaptée à la gestion des transactions en temps réel.

## Description

OROPOS permet de gérer efficacement les commandes, afficher les transactions en temps réel et offrir une navigation intuitive pour les utilisateurs. Le projet se concentre sur l’expérience utilisateur et l’optimisation de la performance de l’interface.

---

## Objectifs du projet

- Créer une **interface utilisateur intuitive et réactive**.
- Développer les fonctionnalités principales du frontend :
  - Gestion des commandes (ajout, modification, suppression).
  - Affichage **en temps réel des transactions**.
  - Navigation simple et ergonomique.
- Assurer une **collaboration continue** via Git pour le partage du code et la synchronisation des membres de l’équipe.
- Proposer des améliorations pour **optimiser l’expérience utilisateur et la performance**.

---

## Fonctionnalités principales

1. **Gestion des commandes**
   - Ajout, modification et suppression des commandes.
   - Suivi des commandes en cours et terminées.

2. **Tableau de bord en temps réel**
   - Visualisation instantanée des transactions.
   - Mise à jour dynamique sans rechargement.

3. **Navigation intuitive**
   - Menu clair et accessible.
   - Accès rapide aux fonctionnalités clés.

---

## Technologies utilisées

- **Frontend** : Angular, TypeScript, HTML, CSS  
- **Versioning / Collaboration** : Git  
- **Concepts clés** : UX/UI design, Responsive Design, Réactivité temps réel
